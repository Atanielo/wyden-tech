// Aula 2 - Instalando o GLPI

Nessa aula, vamos fazer a instação do sistema GLPI em um
"Debian 8". 

Então o primeiro passo é instalar todas as dependências necessárias:

apt-get update
apt-get dist-upgrade -y
apt-get install vim curl apache2 ufw -y

Agora, vamos colocar o nosso IP ou hostname no apache.conf
vim /etc/apache2/apache2.conf

ServerName Nome_ou_IP_do_Servidor

Agora, vamos testar o nosso Apache e liberar ele em nosso
firewall:

apache2ctl configtest

Agora, vamos instalar o MySQL, ou usar o mesmo do Zabbix:

apt-get install -y mysql-server

mysql -u root -p

mysql> CREATE DATABASE glpiweb;

mysql> CREATE USER 'glpi'@'localhost' IDENTIFIED BY 'glpiteste';

mysql> GRANT ALL PRIVILEGES ON glpiweb.* TO 'glpi'@'localhost';

mysql> FLUSH PRIVILEGES;

Agora, vamos instalar o PHP e suas bibliotecas:

# apt-get install ca-certificates apache2 libapache2-mod-php5 php5-cli php5 php5-gd php5-imap php5-ldap php5-mysql php-soap php5-xmlrpc zip unzip bzip2 unrar-free php5-snmp php5-curl -y

# systemctl restart apache2

Agora, vamos instalar GLPI na versão 9.1

# cd /var/tmp

# wget -c https://github.com/glpi-project/glpi/releases/download/9.1/glpi-9.1.tar.gz

# tar xvzf glpi-9.1.tar.gz -C /var/www/html

# chown www-data:www-data -R /var/www/html/glpi

# chmod 777 /var/www/html/glpi/config /var/www/html/glpi/files

Agora, vamos fazer umas configurações em nosso sistema:

# vim /etc/apache2/conf-available/glpi.conf

<Directory /var/www/html/glpi>
AllowOverride All
</Directory>
<Directory /var/www/html/glpi/config>
Options -Indexes
</Directory>
<Directory /var/www/html/glpi/files>
Options -Indexes
</Directory>

Agora, vamos habilitar no apache:

# a2enconf glpi

# service apache2 reload

Pronto! Agora é só entrar no painel Web de seu servidor, ex:

192.168.1.250/glpi e concordar com os termos da licença GNU
e depois colocar o endereço de seu banco de dados e depois colocar o nome de usuário e senha criados para ele:

localhost

glpi

glpiteste

Depois, ponha o usuário e senha:

glpi/glpi
